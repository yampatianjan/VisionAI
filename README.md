# VisionAI - Cloud Image Intelligence Platform

A cloud-based image intelligence platform built with Flask and AWS Rekognition.

VisionAI is a cloud-based image intelligence platform built using Flask and AWS Rekognition. The application enables users to upload images and receive AI-powered insights including object detection, facial attribute analysis, image metadata extraction, and downloadable PDF reports through a modern web interface.

---


## Features

- ✅ Object Detection
- ✅ Face Analysis
- ✅ Drag & Drop Image Upload
- ✅ PDF Report Generation
- ✅ Image Metadata Analysis
- ✅ Responsive User Interface
- ✅ AWS Rekognition Integration

## Installation

Clone the repository

```bash
git clone https://github.com/yampatianjan53-create/VisionAI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Configure AWS credentials

```bash
aws configure
```

Run the application

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
